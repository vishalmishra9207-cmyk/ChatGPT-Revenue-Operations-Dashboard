# 📊 ChatGPT Global Operations Dashboard

This project analyzes 1M+ records of AI usage data to track revenue, cost, and user engagement trends.

## 🖼️ Dashboard Preview
![Dashboard Screenshot](dashboard_preview.png)

## 🛠️ Skills Showcased
- **Data Transformation:** Power Query for cleaning and formatting 1M+ rows.
- **Advanced DAX:** Created measures for `Total Profit`, `Profit Margin %`, and `Average Engagement`.
- **UI/UX Design:** Implemented a Dark Mode theme with rounded corners for a premium feel.

## 📈 Key Insights
- Identified Top 5 revenue-generating countries.
- Compared GPT-4 vs GPT-3.5 performance.
- Correlated session duration with message frequency
